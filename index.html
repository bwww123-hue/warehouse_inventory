Firebase 연동 설정

Firebase Realtime Database를 연결하면 여러 기기·여러 사람이 재고 데이터를 실시간으로 공유할 수 있습니다.

1
Firebase 콘솔에서 새 프로젝트를 만드세요.
2
빌드 → Realtime Database를 생성하고, 규칙(Rules)을 아래처럼 설정하세요.
{ "rules": { ".read": true, ".write": true } }
3
프로젝트 설정 → 앱 추가(웹)에서 databaseURL을 복사해 아래에 붙여넣으세요.
올바른 Firebase Database URL을 입력해주세요.
